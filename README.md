Movie Genre Classification using Machine Learning

Objective:
Build a machine learning model to predict the genre of a movie based on its plot description.

Dataset:
The dataset contains movie titles, genres, and descriptions.

Methodology:
1. Data preprocessing
2. Text vectorization using TF-IDF
3. Multinomial Naive Bayes classifier
4. Model evaluation using accuracy and confusion matrix

Libraries Used:
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

Result:
The model successfully predicts movie genres from textual descriptions with around 55–60% accuracy.
